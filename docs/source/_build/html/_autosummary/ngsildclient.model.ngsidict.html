This class is a custom dictionary that backs an entity.

NgsiDict is used to build and hold the entity properties, as well as the entity’s root. It’s not exposed to the user but intended to be used by the Entity class. NgsiDict provides methods that allow to build a dictionary compliant with a NGSI-LD structure.

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
